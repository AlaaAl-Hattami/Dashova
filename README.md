# 🖥️ **DashOva - Admin Dashboard**

**DashOva** is an intuitive and secure admin dashboard designed with modern technologies. This application allows admins to sign in and out securely using JWT (JSON Web Tokens) for authentication. Built with Node.js, Express, TailwindCSS, and EJS, it provides a simple yet powerful interface for managing user data.

---

## 🔑 **Key Features:**

- 🔐 **Secure Login & Logout** using JWT
- 🧑‍💻 **Admin Dashboard Interface**
- 🚀 **Built with Node.js, Express, TailwindCSS, and EJS**
- 🛠️ **Customizable user interface with TailwindCSS**

---

## 💻 **Technologies Used:**

- **Frontend**: TailwindCSS, EJS
- **Backend**: Node.js, Express
- **Authentication**: JWT (JSON Web Tokens)
- **Database**: Optional (can integrate MongoDB, PostgreSQL, or others)
